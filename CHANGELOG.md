## 0.9.9 (2025-12-14)

First public release of CodexUse Desktop.

### Highlights

- Profile switching across multiple accounts (tray or window)
- Pick an agent per profile: Codex CLI or Claude Code
- Rate limit telemetry, with optional auto-roll to a fresh profile
- Local-first config and data in `~/.codex`
- MCP server management from the UI

### Downloads

- macOS: `CodexUse-mac-arm64.dmg` (plus `.zip` for auto-updates)
- Linux: `CodexUse-linux-arm64.AppImage`

### Known limitations

- Windows build is not published yet.
